  Thermal/mechanical analysis
 
  Solid cylinder/taper/sphere—-temperature loading. 
      Features:
      - Axial symmetry for model dimension reduction.
      - Thermal-strain effect.
      - Extrapolation of quadrature point data to nodes.
 
  Solid cylinder/taper/sphere—-temperature loading; quadratic brick mesh. 
      Features:
      - Fully three-dimensional model.
      - Use of normal-spring sliding constraints.
      - Thermal-strain effect.
      - Extrapolation of quadrature point data to nodes.
      - Using super convergent patch recovery (SPR) for stress postprocessing.
 
  Cooling shock loading of porcelain plate (plane-strain model). 
      Features:
      - Transient cooling of porcelain plate.
      - Thermal-strain effect.
      - Thermal and mechanical simulation in lockstep in the time loop.
      - Graphical postprocessing.
